# Parth Prajapati Portfolio

A modern futuristic developer portfolio built with React, TypeScript, Tailwind CSS, and Framer Motion featuring immersive UI, smooth animations, glassmorphism effects, and responsive design.

## 🚀 Live Demo

🌐 https://parth-portfolio-3dm44z69e-parth20098-8349s-projects.vercel.app

## ✨ Features

- Modern futuristic UI/UX
- Fully responsive across all devices
- Smooth animations with Framer Motion
- Interactive hero section
- Glassmorphism effects
- Dynamic project showcase
- Certifications & experience timeline
- Optimized performance
- Dark cinematic theme

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite
- Lucide React

## 📦 Installation

bash
git clone https://github.com/YOUR_USERNAME/parth-portfolio.git

cd parth-portfolio
